🧮 Calculator Python Package

Python 3.8+ MIT License GitHub Actions

A demonstration Python package for learning GitHub Actions CI/CD workflows. This project showcases automated testing, building, and deployment processes.

🎯 Workshop Goal: Learn to set up automated build pipelines using GitHub Actions in under 1 hour!

✨ Features

Basic Operations

Add, subtract, multiply, divide with error handling

Advanced Math

Power and square root operations using NumPy

API Integration

Fetch random numbers from external API with fallback

Full Test Coverage

Comprehensive pytest suite with mocking

🚀 Quick Start

Installation

git clone https://github.com/USERNAME/REPO.git
cd gh-actions-python-1hr
pip install -r requirements.txt

Usage

# Run the calculator
python calculator.py

# Run tests
pytest test_calculator.py

# Build package
python -m build

🔧 GitHub Actions Workshop

This repository demonstrates a simple CI/CD pipeline with the following stages:

  1. Test Stage: Runs pytest to validate code functionality
  2. Build Stage: Creates Python package distributions
  3. Deploy Stage: Publishes documentation to GitHub Pages

Workshop Learning Objectives

  • Understand GitHub Actions workflow syntax
  • Set up automated testing
  • Create build artifacts
  • Deploy static content to GitHub Pages
  • Use job dependencies and conditional execution
